# Simple Pilot Command

Simple Pilot Command is an Arma 3 mod which allows players to give movement Commands to a helicopters pilot as with ground vehicles.

## planned features:

- adjusting speed
- adjusting heading
- adjusting height
- loitering
- auto align
- panic egress
- gunner/pilot remote control switch
- key binding config
